MLMClass

Overview

A multi-level class is defined to have a level. A multi-level class has a meta-class which must occur at the next higher level. A class at level 1 will have objects as instances at level 0. The class specifies properties, each of which has a level number indicating when they become slots in (potentially indirect) instances of the class.
