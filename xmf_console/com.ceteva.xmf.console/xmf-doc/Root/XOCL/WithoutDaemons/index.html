WithoutDaemons

Overview

The without daemons expression is used to turn daemons off for the evaluation of an expression in the context of an object. Often a daemon will fire and the code that handles the daemon should not cause recursive daemon invocation. This form can be used to safely turn off daemons temporarily for the scope of the handler. Constructor: WithoutDaemons(object,body) The object and body should be expressions. Daemons are switched off for the scope of the evaluation of the body.
