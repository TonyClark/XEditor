Syntax

Overview

A syntax class defines a special type of performable element that evaluates to itself instead of to a performable element whose semantics is already known. The benefit of this is that the structure of the syntax element is not lost when the language feature is evaluated. The consequence of desugaring to itself is that the element must specify its own semantics.
