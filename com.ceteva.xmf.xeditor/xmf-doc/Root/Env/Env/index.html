Env

Overview

The class Env is abstract and is the superclass of all environment implementation classes. Use Env as the type for all environments.

Parents Object
Subclasses Pair ,NullEnv

Operations
add(e:Env):Env Merge two environments to produce a new environment. Note that the left hand environment shadows bindings in the right hand environment with the same name.
bind(name:String,value:Element):Env Adds an association between the key and the value. Returns a new environment which is a copy of the receiver exten-ded with the association.
binding(key:Element):Element Returns the most recently defined binding for the key or null.
binds(key:Element):Boolean Returns true when the key is associated in the environment otherwise returns false.
