Handler

Overview

A dispatch handler defines the code that is performed when a particular type of value is walked. The form of a handler is: @Handler in (,,) en-d here the handler is invoked when a value of the given type is walked. The DISP expression defines the dispatcher object to which the handler is added. The names VALUE ARG and ENCOUNTER are bound to the walked object, the walker argument (often null) and whether this is a subsequent encounter or not.
