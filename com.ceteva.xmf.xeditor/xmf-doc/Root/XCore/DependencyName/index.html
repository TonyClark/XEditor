DependencyName

Overview

A depen-dency name is a name that can be associated with a depen-dency and can represent any textual information relating to that depen-dency. It can be associated to the source, target or the depen-dency itself, and can be set as modifiable or not (this will determine whether or not the diagram labels will be editable). This class is not itself expected to be sub-classed. Instead, the semantics of setting the values of these names is specified by over-riding the 'setNameValue' in sub-classes of Depen-dency.
